# RVM

A simple rust 'virtual machine'.

In fact, this is just a userspace program execution.

# Dependencies

* cargo
* rustc

# Build

Assured by our servitor `cargo`.

`cargo build`

# Run

`caro run $ToExec`

where `$ToExec` is the executable you want run.

# About

Handled architecure:

In the scope:
  * `x86`
  * `x64`
